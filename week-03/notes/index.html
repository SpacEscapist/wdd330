Week 03 Notes


  • OBJECTS
  • An object's method can be called either by using dot or bracket notation.
    - dot notation: superman.fly();
    - bracket notation: superman["fly"]();
  • You can iterate over an object's key's using "Object.keys()" method.
    for(const key of Object.keys(superman)) {
    console.log(key) };
  • You can iterate over an object's values's using "Object.values()" method.
    for(const value of Object.values(superman)) {
    console.log(value) };
  • You can return an array of key-value pairs using "Object.entries()" method.
    for(const [key,value] of Object.entries(superman)) {
    console.log.(`${key}: ${value}`); }
  • Objects can be used as parameters in to functions. This allows arguments to be entered in any order, or even omitted.
  • JSON is a string representation of a JavaScript object. There are a few differences between JSON and JavaScript object:
    1. Property names must be double-quoted.
    2. Permitted values are double-quoted strings, numbers, true, false, null, arrays, and objects.
    3. Functions are not permitted values.
  • You can "stringify()" a JavaScript object, which returns the object as a string of text. If you do so, any methods of that object will simply be ignored, and not show up in the text.


  • DOCUMENT OBJECT MODEL
  • While JavaScript is the most used language with the DOM, the DOM is actually language agnostic
  • The DOM contains an object called "document". This document object allows us to perform various methods on it, such as...
    - document.getElementById()
    - document.querySelector()
    - document.createElement("li")
  • You can use the "classList" property to add a new class to an element. For example...
    classList.add("new-class-name")
  • You can access an element's attributes by using the .getAttribute() method.
    You can add an attribute to an element by using the .setAttribute() method.
  • The .innerHTML property will allow us to insert raw HTML directly into the DOM.
    Whereas, .textContent will only allow us to insert text. Any HTML added will only display in a raw text form.


  • EVENTS
  • A full list of Events can be found HERE
  • Event listeners can be added using ".addEventListener()"
  • Event listeners can take a total of 3 arguments, althought it's most common to only use 2 arguments:
    - First argument: The type of event, such as "click"
    - Second argument: A callback function which is invoked when the event occurs
    - Third argument: A boolean value that specifies whether capturing should be used or not. Default is set to false
  • The more commonly used Events are:
    - Mouse Events
    - Keyboard Events
    - Touch Events (for touch-screen devices)
  • Some of the common Mouse events:
    - click
    - mousedown
    - mouseup
    - dblclick
  • Some of the common Keyboard events:
    - keydown
    - keypress
    - keyup
  • Some of the common Touch events:
    - touchstart
    - touchend
    - touchmove
    - touchenter
    - touchleave
    - touchcancel
